# 🚨 Smart Earthquake Emergency Bag Assistant

## 📝 Description
An emergency program designed to help people prepare their survival kits and manage essentials during earthquake situations.

## ✨ Features
* Inventory Management: Track essential items in your bag.
* Smart Reminders: Alerts for expiring items (food/water).
* Emergency Guide: Quick tips for earthquake safety.

## 🛠️ Built With
* Python 🐍
* Tkinter (Desktop GUI)
* SQL (Database management)

## 🚀 How to Run
1. Download the code.
2. Run main.py.
